# 1st_Java_CaseStudy
Bank Automation System (Java)
-> A mini project to automate core banking operations for Bank.
-> Handles over-the-counter transactions and account lifecycles.
-> Supports Savings, Salary, Current, and Loan accounts.
-> Implements interest calculation based on account type.
-> Tracks inactive accounts and enforces rules (e.g., minimum balance, overdraft, loan repayment).
-> Generates end-of-day transaction reports.
